# Ubuntu Dev

A quick and convenient Ubuntu dev environment to hack stuff out from my Mac.

## Usage

Once:
```bash
$ ./ub build
```

Start:
```bash
$ ./ub start
ensuring docker
/usr/local/bin/docker
starting dev shell
mapping /Users/bryan/Projects/ubdev/..:/projects
root@ubdev:/projects#
```
